# Thermal-EX

In this project, shell will print the logo of Arknights and Thermal-EX. Ready to add other functions.
## How to use

```cmake
mkdir build
cd build
cmake .. -DCMAKE_BUILD_TYPE=Debug
# cmake .. -DCMAKE_BUILD_TYPE=Release
make -j
./lightWithLogo
```

For clean: in the /build , `make clean`
